# Halo! Kami Rasi Code 👋

Kami adalah tim pengembang perangkat lunak dan solusi teknologi yang berbasis di **Yogyakarta**. Rasi Code berfokus pada pembangunan produk digital yang andal, efisien, dan inovatif untuk membantu bisnis berkembang di era digital.

---

## 🚀 Tentang Kami

Di Rasi Code, kami percaya bahwa kode yang baik adalah fondasi dari bisnis yang hebat. Kami menggabungkan kreativitas, keahlian teknis, dan pemahaman bisnis untuk menghadirkan solusi kustom mulai dari landing page, company profile, hingga sistem informasi berskala besar.

* 📍 **Lokasi:** Yogyakarta, Indonesia
* 🌐 **Layanan:** Pengembangan Web, Aplikasi Seluler, & Solusi IT Perusahaan
* ✉️ **Hubungi Kami:** admin@rasicode.com
